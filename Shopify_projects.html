YE-WEB

YE-WEB is a fully custom Shopify theme The primary goal of this project was to create a high-performance website for selling event tickets. The theme consists of 12 to 15 fully responsive pages, including a uniquely designed and highly functional custom product page tailored specifically for ticket sales. I built the entire theme from scratch using HTML, CSS, JavaScript, Liquid The UI/UX design was provided via Canva, and I was responsible for accurately translating it into a fully functional, pixel-perfect Shopify theme. I ensured cross-browser compatibility, mobile responsiveness, and performance optimization. This project strengthened my expertise in Shopify theme development and my ability to implement complex frontend requirements based on predefined designs.

Hanami

Hanami is a fully custom, single-page Shopify theme developed for a restaurant to manage reservations and bookings online. The design was provided via Figma, which I carefully replicated into a responsive, high-performance Shopify theme. The project includes 5 unique custom sections, 2 fully functional forms for table reservations and contact inquiries, and dynamic content integration using Liquid. I implemented clean, maintainable code using HTML, CSS, JavaScript, and Liquid, ensuring smooth user experience across all devices. The theme is optimized for both desktop and mobile

Chill-N Pint

Chill-N Pint Club is a custom Shopify theme built for a global brand that delivers nitrogen ice cream worldwide. The product page features an advanced setup with 400 variants — 4 pints, each offering 100 unique combinations. The page includes multiple customizable sections where users can select a limited number of flavors and toppings from each group. After making their selections, customers can proceed to add the product to the cart. Before checkout, the form prompts users with a question regarding any food allergies. I developed this functionality using custom forms with HTML, CSS, JavaScript, and Liquid, ensuring a seamless and interactive user experience.
