price oye

Price-Oye is a fully custom WooCommerce WordPress theme. It's a side project of mine focused on WordPress theme development. All pages and functionalities are fully customized using HTML, CSS, Tailwind, PHP, and WordPress core functions. The entire content is dynamic using ACF (Advanced Custom Fields). AJAX is implemented on the product page, and the shop page includes custom filters. The login, registration, main cart page, and category pages are all custom-built.
